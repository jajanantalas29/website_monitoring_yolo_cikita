📌 PRASYARAT (Pastikan Sudah Ada di Laptop)
1.Laragon (Untuk menjalankan MySQL Database).
2. Python 3.10 (Wajib versi 3.10 agar AI tidak error).
3. Node.js & Composer (Untuk menjalankan website Laravel).
4. Cloudflared

3) Setup AI Server (Terminal 2)
1. Buka folder Model AI.
2. Buka Terminal (CMD) di dalam folder tersebut.
3. Buat "wadah" virtual dan aktifkan dengan perintah ini:

venv\Scripts\activate (Pastikan muncul tulisan (venv) hijau di sebelah kiri CMD).

4. Install semua bahan AI (Butuh koneksi internet stabil & lumayan lama):

pip install -r requirements.txt

🚀 BAGIAN B: Cara Menjalankan Sistem Sehari-hari
(Langkah ini yang kamu lakukan setiap kali mau mendemokan/membuka skripsi).

Pastikan Laragon sudah menyala, lalu siapkan:

🟢 1) Menyalakan Otak AI (FastAPI)

1. Buka CMD di folder Model AI.
2. Ketik: venv\Scripts\activate (lalu Enter).
3. Ketik: uvicorn api:app --host 0.0.0.0 --port 8001 --reload (lalu Enter).
4. Biarkan terminal ini terbuka.

🔵 2) Menyalakan Website (Laravel)

1. Buka CMD baru di folder Website Laravel (terminal di dalam VS Code).
2. Ketik: php artisan serve --host 0.0.0.0 --port 8000 (lalu Enter).
3. Biarkan terminal ini terbuka.

🟡 3) Menyalakan Desain (Tailwind Vite)

1. Buka CMD baru lagi di folder Website Laravel (terminal di dalam VS Code).
2. Ketik: npm run dev (lalu Enter).
3. Biarkan terminal ini terbuka.
4. Jika tampilan menjadi berantakan, maka tidak usah npm run dev
